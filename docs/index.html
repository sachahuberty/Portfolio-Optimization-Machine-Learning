The Challenge

Why Naive ML Fails in Finance

Financial markets are hostile, non-stationary environments. Standard ML practices introduce systematic biases that create the illusion of alpha — not real performance.

01Label Leakage
Naive:Fixed 1-month return horizon
Ours:Triple Barrier Method — volatility-adaptive, path-aware
02Survivorship Bias
Naive:Today's index projected backwards
Ours:Historical reconstitution: 801 tickers incl. ~298 delisted
03Look-Ahead Bias
Naive:Fundamentals at fiscal quarter-end date
Ours:SEC acceptance timestamps via edgartools (PIT)
04Overfitting
Naive:Standard k-fold cross-validation
Ours:CPCV: 36 folds, purged + 5-day embargo
05Uncalibrated Sizing
Naive:Single model for direction + size
Ours:Meta-labeling: direction and conviction decoupled
06Beta as Alpha
Naive:Raw model predictions
Ours:Per-date neutralization (Moore-Penrose pseudo-inverse)
